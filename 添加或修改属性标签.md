# 添加或修改属性标签

## 1. API 描述：添加或修改属性标签


## 2. 调用方法

> post /api/tag-save

## 2. 请求参数

参数 | 解释 | 类型 | 是否必须 | 示例数据
:---:|:---|:---:|:---:|:---



## 3. 响应状态

状态码 | 说明
:---:|:---
200 | 操作成功


## 4. 响应数据

参数 | 解释 | 类型 | 是否必须 | 示例数据
:---:|:---|:---:|:---:|:---



## 5. 示例
{url}/api/tag-save

### 请求数据：

备注：当有id，为修改信息，没有id为添加信息。
所有信息可选填

```json
{
    "id":2
	"name":"互联网"
}
```


### 响应结果：


```json
{
    "code": 200,
    "msg": "success",
    "data": {
        "id": 2,
        "name": "互联网"
    }
}
```
