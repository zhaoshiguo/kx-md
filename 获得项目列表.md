# 获得项目列表接口

## 1. API 描述：获得所有项目列表，可以根据关键字模糊查找


## 2. 调用方法

> get /api/project-list

## 2. 请求参数

参数 | 解释 | 类型 | 是否必须 | 示例数据
:---:|:---|:---:|:---:|:---
page | 当前页数 | int | 否 | 默认（1）
num | 当前条数 | int | 否 | 默认（20）
kw | 模糊查找关键字 | date | 否 | 默认（""）


## 3. 响应状态

状态码 | 说明
:---:|:---
200 | 操作成功


## 4. 响应数据

参数 | 解释 | 类型 | 是否必须 | 示例数据
:---:|:---|:---:|:---:|:---



## 5. 示例
{url}/api/project-list?kw=
